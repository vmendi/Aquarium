Bet Aquarium
===================

Create and nurture your own aquarium while winning money!



Installation
------------
*npm install*
(from the app directory)


Configuration
------------

#### config/configs/base.js

* Update *game_id*

#### config/configs/(real|sandbox).js

* Update *api_key*
* Update *api_secret*

Start the server
------------
*NODE_ECONOMY=(sandbox|real) node server.js*
